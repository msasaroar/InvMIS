# ২️⃣ চূড়ান্ত টেস্টিং Guide – InvMIS API

### Step 1: Database Migrations

```bash
cd InvMIS.Infrastructure
dotnet ef migrations add InitialCreate
dotnet ef database update
```

* নিশ্চিত করো `appsettings.json` এ PostgreSQL connection string ঠিক আছে।
* Database নাম হবে **InventoryDB**

---

### Step 2: Run API

```bash
cd InvMIS.API
dotnet run
```

* বা Visual Studio তে **F5** চাপো।

---

### Step 3: Swagger UI / API Testing

1. Browser এ যাও: `https://localhost:5001/swagger`
2. Controllers সব দেখাবে: Product, Category, Supplier, Stock, Auth
3. **Authorize** button দিয়ে JWT token সেট করো

   * Token পাওয়ার জন্য `/api/Auth/login` POST request:

     ```json
     {
       "username": "admin",
       "password": "Admin123!"
     }
     ```
4. Token দিয়ে সব **Authorized endpoints** (POST/PUT/DELETE) access করতে পারবে

---

### Step 4: CRUD Testing Example

**Product API**:

* GET: `/api/Product` → সব products দেখাবে
* POST: `/api/Product` → নতুন product add (Admin only)
* PUT: `/api/Product/{id}` → product update (Admin only)
* DELETE: `/api/Product/{id}` → product delete (Admin only)

**Category, Supplier, Stock** একইভাবে পরীক্ষা করা যাবে

---

### Step 5: Admin User Seed Verification

* DbSeeder automatically Admin user তৈরি করে
* Default Admin username/password:

  ```text
  username: admin
  password: Admin123!
  ```
